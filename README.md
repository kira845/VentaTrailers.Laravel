# 🚛 Sistema Web de Venta de Trailers

Aplicación web desarrollada para la gestión y visualización de productos de una empresa dedicada a la venta de trailers, incluyendo funcionalidades de catálogo, contacto y administración.

---

## 🧩 Características principales

* Gestión de productos (CRUD)
* Visualización de catálogo por categorías
* Sistema de autenticación (login)
* Panel de administración
* Interfaz orientada a la venta y contacto con clientes
* Diseño adaptable para visualización web

---

## 🛠️ Tecnologías utilizadas

* Backend: Laravel 
* Base de datos: MySQL 
* Frontend: HTML, CSS
* Otros: Bootstrap
